How to use it:

1. Add script src to "extends":

`.stylelintrc.js`
---
```js
module.exports = {
  extends: [
    './index.js',
  ],
```

2. Feel free to use it in your projects!

How to use it:

1. Copy the plugin file into your Nuxt project under the `server/plugins` directory
2. Start your Nuxt server:
```
npm run build
npm run dev
```
